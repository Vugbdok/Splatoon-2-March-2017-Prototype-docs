# How to run the Splatoon 2 (March Prototype) on the Nintendo Switch
> [!important]
> I **WILL NOT** provide the repacked NSP. You **MUST** find it your self.

## What you will need
- A [modded](https://switch.hacks.guide/) Nintendo Switch on Horizon os 21.2.0 (or lower) and Atmosphère v1.10.2+
- The [Sys-patch](https://github.com/impeeza/sys-patch/releases) module for installing the NSP
- The repacked NSP file for the Splatoon 2 (March Prototype)
- The Atmosphère [patches](/assets/Patches.zip) for the Splatoon 2 (March Prototype)
- Some type of NSP installer like [Goldleaf](https://github.com/XorTroll/Goldleaf/releases)
> [!note]
> I will only show Goldleaf for the guide.
# Installing the NSP

I will show you how to install the Splatoon 2 (March Prototype) NSP using Goldleaf.
> [!CAUTION]
> Modifying the system files can result in a **permanent brick** if done incorrectly. I suggest you **back up your system files** before you install the game.

Open a game and select a user. Hold <img src="/assets/Bin/BIN_R.png" alt="Button R" width="25" valign="middle" /> when selecting.

<img src="/assets/screenshots/How2/Useracc.jpg" alt="dummy" height="365" />

From the hbmenu, open **Goldleaf**.

<img src="/assets/screenshots/How2/Glhbmenu.jpg" alt="dummy" height="365" />

In Goldleaf, go to **Explore content**.

<img src="/assets/screenshots/How2/Gl.jpg" alt="dummy" height="365" />

Then go to **SD card**.

<img src="/assets/screenshots/How2/Glsdhover.jpg" alt="dummy" height="365" />

Find and open the Splatoon 2 (March Prototype) NSP on your SD card.

<img src="/assets/screenshots/How2/Glnsphover.jpg" alt="dummy" height="365" />

Select **Install**.

<img src="/assets/screenshots/How2/Glinstallnsp.jpg" alt="dummy" height="365" />

You **MUST** install it to your SysNAND or emuMMC. Select **Console memory**.

<img src="/assets/screenshots/How2/Glinstallnand.jpg" alt="dummy" height="365" />

When done, press <img src="/assets/Bin/BIN_+.png" alt="Button +" width="18" valign="middle" /> to go back to the hbmenu, and then open **USB File Transfer**.

<img src="/assets/screenshots/How2/Usbfthbmenu.jpg" alt="dummy" height="365" />

When you see this screen, connect the Switch to your PC or Mac and copy the patches (atmosphere, and switch) to the root of your SD card. Make sure you merge the folders and not overwrite them.

<img src="/assets/screenshots/How2/Usbft.jpg" alt="dummy" height="365" />

Now close it, return to the Switch HOME Menu, and launch the Splatoon 2 (March Prototype) game.

<img src="/assets/screenshots/How2/S2phome.jpg" alt="dummy" height="365" />

ENJOY!

<img src="/assets/screenshots/How2/S2pintro.jpg" alt="dummy" height="365" />
